Comece sempre pelo edital. O sistema depende disso para montar checklist e foco correto.

  2. Envie o primeiro simulado com: prova bruta, gabarito oficial e suas respostas.
  3. Atualize o histórico e revise os padrões de erro após cada correção.
  4. Repita o ciclo semanalmente: quanto mais simulados, melhor a detecção de vícios e foco de estudo.

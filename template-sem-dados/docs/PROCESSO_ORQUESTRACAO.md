# Processo de Orquestracao

## Fluxo recomendado
1. Usuario resolve prova.
2. Usuario fornece respostas + gabarito.
3. Atualize os arquivos .md e backend/data.json.
4. Dashboard reflete progresso.

## Politica de dados
- Este template nasce sem dados pessoais.
- Cada usuario popula seus proprios simulados.
